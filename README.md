# Online Sales Analysis

Ovaj projekat simulira sistem za upravljanje proizvodima i korpom kupca
u online prodavnici, koristeći Python OOP principe i Git verzionisanje.

## Klase
- Product – model proizvoda
- ProductManager – upravljanje inventarom
- Cart – korpa kupca

## Funkcionalnosti
- Dodavanje, prikaz i uklanjanje proizvoda
- Računanje ukupne vrednosti inventara
- Upravljanje korpom i iznosom za naplatu

## Tehnologije
- Python
- Git & GitHub
